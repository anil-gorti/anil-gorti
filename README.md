## Current Projects

- 🏃 [**Strava Runner Profile**](https://github.com/anil-gorti/strava-runner-profile) — Strava OAuth + multi-agent LLM pipeline to build rich runner profiles
- 📜 [**Athlete Race History**](https://github.com/anil-gorti/athlete-race-history) — Standalone race history module for athlete platforms — Next.js + FastAPI, no Strava dependency
- 🤖 [**Three-Agent Runner Profiler**](https://github.com/anil-gorti/three-agent-runner-profiler) — Three-agent AI pipeline that builds personalised runner profiles from Strava & Gmail data
- 🗂️ [**Race Event Registry Agent**](https://github.com/anil-gorti/race-event-registry-agent) — Autonomous agent that scrapes race-timing platforms and normalises events into a unified schema
- 🕷️ [**Race Results Scraper**](https://github.com/anil-gorti/race-results-scraper) — Python agent that discovers race results pages, intercepts SPA API calls, exports structured CSVs
- 🇮🇳 [**India Race Registry**](https://github.com/anil-gorti/india-race-registry) — Canonical race registry from India's 5 major timing companies (2017–present)
- 🏋️ [**Hyrox Timekeeper App**](https://github.com/anil-gorti/Timekeeper-App-for-Hyrox) — Volunteer timekeeper app for Hyrox race events; tracks athletes through 12 activities
- 🔗 [**Athlete Onboarding Hub**](https://github.com/anil-gorti/athlete-onboarding-hub) — AI-powered onboarding: Strava OAuth + Gmail parsing + race scraper into verified race profiles
- 🏊 [**Endurance Club OS**](https://github.com/anil-gorti/Endurance-Club-OS) — Admin dashboard & club website for endurance sports clubs
- 🧑‍🤝‍🧑 [**Sports Club Onboarding**](https://github.com/anil-gorti/sports-club-onboarding) — Self-serve sports club creation & onboarding — Next.js, Clerk, Supabase, Prisma
- 🎽 [**Club Coaching API**](https://github.com/anil-gorti/club-coaching-api) — Role-based API for sports clubs: members, coaches, athletes, consent & coaching relationships
- 🧬 [**Agentic Athlete Profile**](https://github.com/anil-gorti/agentic-athlete-profile) — Three-agent LLM pipeline that transforms raw runner data into a structured athlete profile
- 🕐 [**Race Timing API Client**](https://github.com/anil-gorti/race-timing-api-client) — Python client for querying race timing APIs — fetch events, results, and athlete data
- 📸 [**Athlete Photo Uploader**](https://github.com/anil-gorti/athlete-photo-uploader) — Admin photo uploader for athlete platforms
- 🏁 [**Race Results Agent**](https://github.com/anil-gorti/race-results-agent) — Agentic pipeline extracting structured race results from 6 timing platforms
